<feature>
  <name>useCalendar Composable</name>
  <files>src/composables/useCalendar.ts, src/composables/useCalendar.spec.ts</files>
  <behavior>
    The composable should accept a year and return an array of 12 month objects.
    Each month object should contain:
    - `name` (string)
    - `days` (array of Day objects)
    - `firstDayOffset` (0-6, starting from Monday)

    Day objects should contain:
    - `date` (Date object)
    - `dayOfMonth` (number)
    - `isWeekend` (boolean)

    Verification cases:
    - February 2024 has 29 days (leap year).
    - January 1st, 2026 is a Thursday.
    - Each month array has exactly the number of days belonging to that month.
  </behavior>
  <implementation>
    Use `date-fns` functions: `eachMonthOfInterval`, `eachDayOfInterval`, `startOfMonth`, `endOfMonth`, `getDay`.
    Ensure Monday is the first day of the week (shift Sunday from 0 to 6).
  </implementation>
</feature>
